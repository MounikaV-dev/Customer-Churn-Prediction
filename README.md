# Customer Churn Prediction

## Overview
This project predicts whether a customer will churn using machine learning techniques.

## Dataset
Telco Customer Churn dataset from Kaggle.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Steps Performed
- Data Cleaning & Preprocessing
- Exploratory Data Analysis
- Feature Engineering
- Model Training (Logistic Regression)
- Model Evaluation

## Results
- Accuracy: ~79%
- Identified key factors affecting customer churn

## Conclusion
This project demonstrates how data-driven approaches can help businesses reduce customer churn.
